# 设备分类处理功能 - 快速部署指令

## 一键复制命令

### 1. 连接服务器
```bash
ssh root@139.224.200.133
```

### 2. 备份数据库
```bash
pg_dump -U postgres configurable_ops > /tmp/backup_$(date +%Y%m%d_%H%M%S).sql
```

### 3. 进入项目目录
```bash
cd /var/www/nyb_weekly_check_tool_1.0
```

### 4. 运行数据库迁移
```bash
python3 migrate_equipment_processing.py
```

### 5. 重启服务
```bash
sudo systemctl restart review-viewer
sudo systemctl status review-viewer
```

### 6. 查看日志（如有问题）
```bash
tail -f logs/rating_app.log
```

### 7. 重新导入设备数据
```bash
python3 import_equipment_data.py
```

## 如果出现问题

### 回滚数据库
```bash
# 找到备份文件
ls -lh /tmp/backup_*.sql

# 恢复（替换时间戳）
psql -U postgres configurable_ops < /tmp/backup_YYYYMMDD_HHMMSS.sql
```

### 重启服务
```bash
sudo systemctl restart review-viewer
```

### 查看错误日志
```bash
tail -100 logs/rating_app.log
journalctl -u review-viewer -n 50
```
