#### 输入框优化
- 边框加粗(2px)
- 聚焦高亮效果
- 圆角优化(6-8px)

### 4. 视口配置

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
<meta name="apple-mobile-web-app-capable" content="yes">
```

- 禁止缩放(避免误操作)

## 📝 常见问题

### Q1: 输入框点击后页面缩放

**原因**: iOS会自动缩放字体小于16px的输入框

**解决**: 
```css
.reason-input {
    font-size: 16px; /* iOS避免自动缩放 */
}
```

### Q2: 按钮点击区域太小

**原因**: 触摸目标小于44px

**解决**:
```css
.action-btn {
    min-height: 48px; /* 移动端推荐 */
}
```

### Q3: 横屏布局错乱

**原因**: 没有考虑横屏模式

**解决**:
```css
@media (max-width: 768px) and (orientation: landscape) {
    .image-container {
        height: 180px;
    }
}
```

### Q4: 图片加载慢

**原因**: 图片尺寸过大

**解决**:
- 使用lazy loading
- 压缩图片
- 使用CDN

### Q5: 滚动不流畅

**原因**: 动画过多或重绘频繁

**解决**:
```css
* {
    -webkit-overflow-scrolling: touch;
}
```

## 🚀 性能优化

### 1. 减少重绘
- 使用transform代替top/left
- 避免频繁修改DOM
- 使用CSS动画代替JS动画

### 2. 图片优化
- 使用WebP格式
- 实现懒加载
- 设置合适的尺寸

### 3. 代码优化
- 压缩CSS/JS
- 使用缓存
- 减少HTTP请求

### 4. 网络优化
- 启用Gzip压缩
- 使用CDN
- 实现离线缓存
