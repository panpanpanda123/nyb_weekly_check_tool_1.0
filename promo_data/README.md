# 活动参与度数据文件夹

## 文件说明

### 1. 活动参与度文件
- 文件名格式：`活动参与度MM月DD日.xlsx` 或 `活动参与度YYYY年MM月DD日.xlsx`
- 示例：`活动参与度3月9日.xlsx`
- 说明：每天更新，脚本会自动选择最新的文件

### 2. 活动门店表
- 文件名：`3月活动门店表.xlsx`
- 说明：参加3月活动的门店列表，第一列为门店ID

## 数据导入

### 方法1：手动导入
```bash
python3 import_promo_data.py
```

### 方法2：定时任务
```bash
# 每天早上8点自动导入
0 8 * * * cd /var/www/nyb_weekly_check_tool_1.0 && python3 import_promo_data.py >> logs/promo_import.log 2>&1
```

## 数据字段

从活动参与度Excel文件中提取：
- A列：门店ID
- F列：订单量
- I列：权益卡销量
- J列：活动套餐销量
- K列：活动参与度

## 注意事项

1. 文件必须放在此文件夹中
2. 文件名必须包含"活动参与度"
3. 脚本会自动选择最新的文件
4. 每次导入会清空同一日期的旧数据
