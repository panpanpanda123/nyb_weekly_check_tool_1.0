## 🔑 关于SSH密钥

如果你想配置SSH密钥免密登录（可选）：

### Windows用户

1. 打开PowerShell，生成密钥：
   ```powershell
   ssh-keygen -t rsa -b 4096
   ```

2. 上传公钥到服务器：
   ```powershell
   type $env:USERPROFILE\.ssh\id_rsa.pub | ssh root@139.224.200.133 "cat >> ~/.ssh/authorized_keys"
   ```

3. 以后就可以免密登录了

但这不是必须的，用密码登录也完全可以！

## ❓ 常见问题

### Q1: 找不到equipment_status文件夹

**解决**：在服务器上创建
```bash
mkdir -p /var/www/nyb_weekly_check_tool_1.0/equipment_status
```

### Q2: 权限不足

**解决**：确保用root用户登录，或者修改权限
```bash
chmod 755 /var/www/nyb_weekly_check_tool_1.0/equipment_status
```

### Q3: 导入脚本报错

**解决**：查看详细错误信息，可能是：
- 文件格式不对
- whitelist未导入
- 数据库连接失败

### Q4: 连接超时

**解决**：
- 检查服务器IP是否正确
- 检查防火墙是否开放22端口
- 尝试用手机热点连接
