# 📁 文件上传说明

## 方式一：通过网页上传（最简单）⭐

1. **双击运行** `打开上传页面.bat`
2. 浏览器会自动打开上传页面
3. 按照页面提示上传文件：
   - 白名单文件：Excel 格式 (.xlsx)
   - 审核结果文件：CSV 格式 (.csv)

**或者直接访问：** http://139.224.200.133/admin/upload

---

## 方式二：使用 Python 脚本上传

如果网页上传失败，可以使用命令行工具：

```bash
# 安装依赖（首次使用）
pip install requests

# 运行上传脚本
python upload_files.py
```

按照提示选择文件类型并输入文件路径即可。

---

## 方式三：使用 SCP 命令上传（高级）

如果你熟悉命令行，可以直接使用 SCP：

```bash
# 上传白名单文件
scp 白名单.xlsx root@139.224.200.133:/opt/review-result-viewer/data/

# 上传审核结果文件
scp 审核结果.csv root@139.224.200.133:/opt/review-result-viewer/data/
```

然后访问网页管理界面导入数据。

---

## 文件格式要求

### 白名单文件 (.xlsx)
必须包含以下列：
- 门店编号
- 门店名称
- 战区
- 省份
- 城市
- 门店标签（可选）

### 审核结果文件 (.csv)
必须包含以下列：
- 门店编号
- 门店名称
- 检查项名称
- 审核结果（合格/不合格）
- 问题描述（不合格时必填）
- 审核时间
- 图片URL（可选）

## 常见问题

### Q: 上传后看不到数据？
A: 刷新页面，或者检查文件格式是否正确。

### Q: 上传失败怎么办？
A: 
1. 检查网络连接
2. 确认文件格式正确
3. 文件大小不超过 50MB
4. 尝试使用其他上传方式

### Q: 如何更新数据？
A: 重新上传文件会覆盖旧数据。建议先导出备份。
