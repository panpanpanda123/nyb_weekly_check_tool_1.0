# 设备数据导入快速参考

## 命令速查表

| 命令 | 说明 | 使用场景 |
|------|------|----------|
| `python3 import_equipment_data.py` | 导入所有设备 | 全量更新 |
| `python3 import_equipment_data.py --only-pos` | 只导入POS | 每日POS检查 |
| `python3 import_equipment_data.py --only-stb` | 只导入机顶盒 | 周度机顶盒检查 |
| `python3 import_equipment_data.py --clear-pos` | 清空POS数据 | 临时隐藏POS |
| `python3 import_equipment_data.py --clear-stb` | 清空机顶盒数据 | 临时隐藏机顶盒 |

## 数据文件要求

| 文件类型 | 文件名格式 | 何时必需 |
|----------|-----------|----------|
| 在营门店 | `*在营门店*.xlsx` | 总是必需 |
| 收银设备 | `牛约堡集团_点餐设备表_*.xlsx` | 导入POS时 |
| 机顶盒 | `202*.xlsx` | 导入机顶盒时 |

## 典型工作流

### 每日工作流（只检查POS）
```bash
# 1. 上传文件
# - 在营门店.xlsx
# - 收银设备.xlsx

# 2. 导入数据
python3 import_equipment_data.py --only-pos

# 3. 区域经理处理POS异常
```

### 周度工作流（检查机顶盒）
```bash
# 1. 上传文件
# - 在营门店.xlsx
# - 机顶盒.xlsx

# 2. 导入数据
python3 import_equipment_data.py --only-stb

# 3. 区域经理处理机顶盒异常
```

### 全量更新工作流
```bash
# 1. 上传所有文件
# - 在营门店.xlsx
# - 收银设备.xlsx
# - 机顶盒.xlsx

# 2. 导入数据
python3 import_equipment_data.py

# 3. 区域经理处理所有异常
```

## 数据清理规则

| 导入模式 | 清空POS | 清空机顶盒 | 清空处理记录 |
|----------|---------|-----------|-------------|
| 默认（全部） | ✅ | ✅ | ✅ 全部 |
| `--only-pos` | ✅ | ❌ | ✅ 只POS |
| `--only-stb` | ❌ | ✅ | ✅ 只机顶盒 |
| `--clear-pos` | ✅ | ❌ | ✅ 只POS |
| `--clear-stb` | ❌ | ✅ | ✅ 只机顶盒 |

## 服务器快速操作

```bash
# 连接服务器
ssh root@139.224.200.133

# 进入项目目录
cd /var/www/nyb_weekly_check_tool_1.0

# 查看当前数据文件
ls -lh equipment_status/

# 导入数据（选择一个）
python3 import_equipment_data.py              # 全部
python3 import_equipment_data.py --only-pos   # 只POS
python3 import_equipment_data.py --only-stb   # 只机顶盒

# 重启服务
sudo systemctl restart review-viewer

# 查看日志
journalctl -u review-viewer -n 50
```

## 常见错误

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 未找到收银设备文件 | 文件不存在或名称不匹配 | 检查文件名格式 |
| 未找到机顶盒文件 | 文件不存在或名称不匹配 | 检查文件名格式 |
| 未找到在营门店文件 | 文件不存在或名称不匹配 | 检查文件名格式 |
| 数据库连接失败 | 数据库未启动或配置错误 | 检查数据库状态 |
| 导入 0 条记录 | 无符合条件的数据 | 检查过滤条件 |
