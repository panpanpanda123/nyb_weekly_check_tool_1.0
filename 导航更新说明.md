# 导航更新说明

## 📝 更新内容

已在所有页面添加"活动参与度"导航选项卡，用户可以在各个功能之间快速切换。

## ✅ 修改的文件

### 1. HTML模板文件（添加导航按钮）
- `viewer/templates/viewer.html` - 周清审核页面
- `viewer/templates/rating.html` - 门店评级页面
- `viewer/templates/equipment.html` - 设备异常页面
- `viewer/templates/promoratio.html` - 活动参与度页面

### 2. JavaScript文件（添加点击事件）
- `viewer/static/viewer.js` - 周清审核逻辑
- `viewer/static/rating.js` - 门店评级逻辑
- `viewer/static/equipment.js` - 设备异常逻辑
- `viewer/static/promoratio.js` - 活动参与度逻辑

## 🎯 导航结构

所有页面现在都有统一的导航选项卡：

```
📝 周清审核  |  ⭐ 门店评级  |  🔧 设备异常  |  📊 活动参与度
```

点击任意选项卡会跳转到对应页面。

## 🚀 部署步骤

### 1. 在服务器上更新代码
```bash
cd /var/www/nyb_weekly_check_tool_1.0
git pull
```

### 2. 重启服务
```bash
sudo systemctl restart review-viewer
```

### 3. 清除浏览器缓存
- 使用 Ctrl+F5 强制刷新
- 或清除浏览器缓存
- 或使用隐私模式访问

导航按钮使用统一样式：

## 🔧 技术细节

### HTML结构
```html
<div class="function-tabs">
    <button id="reviewTab" class="tab-btn">📝 周清审核</button>
    <button id="ratingTab" class="tab-btn">⭐ 门店评级</button>
    <button id="equipmentTab" class="tab-btn">🔧 设备异常</button>
    <button id="promoTab" class="tab-btn active">📊 活动参与度</button>
</div>
```

### JavaScript事件绑定
```javascript
document.getElementById('promoTab').addEventListener('click', function() {
    window.location.href = '/promoratio';
});
```

## ⚠️ 注意事项

1. **浏览器缓存**：更新后可能需要清除缓存才能看到新的导航按钮
2. **CSS版本**：模板中使用随机版本号防止缓存：`?v={{ range(1, 999999) | random }}`
3. **向后兼容**：所有现有功能保持不变，只是添加了新的导航入口

## 📞 问题反馈

如果导航按钮不显示或点击无反应：
1. 清除浏览器缓存（Ctrl+F5）
2. 检查浏览器控制台是否有JavaScript错误
3. 确认服务已重启：`sudo systemctl status review-viewer`
4. 联系企微：窦(dou)
