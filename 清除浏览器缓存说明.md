# 📱 清除浏览器缓存说明

如果更新后看不到新功能（如"已完成"按钮），需要清除浏览器缓存。

## Chrome 手机版

1. 打开 Chrome
2. 点击右上角 **⋮** (三个点)
3. 点击 **历史记录**
4. 点击 **清除浏览数据**
5. 选择 **缓存的图片和文件**
6. 点击 **清除数据**
7. 刷新页面（下拉刷新）

## 或者使用强制刷新

在浏览器地址栏输入：
```
http://139.224.200.133/?v=2
```

添加 `?v=2` 参数可以绕过缓存。

## 鸿蒙浏览器

1. 点击右下角 **⋯** (更多)
2. 点击 **设置**
3. 点击 **隐私和安全**
4. 点击 **清除浏览数据**
5. 选择 **缓存数据**
6. 点击 **清除**
7. 返回页面刷新

## Safari (iPhone)

1. 打开 **设置** App
2. 找到 **Safari**
3. 点击 **清除历史记录与网站数据**
4. 确认清除
5. 重新打开网页

## 快速测试

如果清除缓存后还是看不到，可以：
1. 使用浏览器的 **无痕模式/隐私模式**
2. 或者用另一个浏览器打开

---

## 开发者选项（高级）

如果你熟悉浏览器开发者工具：

1. 打开开发者工具（F12）
2. 右键点击刷新按钮
3. 选择 **清空缓存并硬性重新加载**
